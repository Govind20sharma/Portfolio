
# Govind's Fullstack Portfolio

This is a full-stack portfolio project built using:
- 💻 Frontend: React + Vite (in `/frontend`)
- 🔧 Backend: Express + Nodemailer (in `/Backend`)

## Scripts

- Frontend: `cd frontend && npm run dev`
- Backend: `cd Backend && npm run start`

## Contact Form
Form sends email to personal Gmail and logs data in `logs/contact_submissions.csv`.
